TEM Program Links
Links to some on-line programs for Transmission Electron Microscopy (TEM) and some SEM programs
(this is not a recommendation for or against any particular program)

Simulations:

  • abTEM :  ab initio TEM python API  (J. Madsen & T. Susi)
  • Casino:  monte carlo SEM
  • calidris: electron crystallography
  • cloudEMAPS : Zuo
  • pyEMAPS : python (M&M 29s1 p.717-718)
  • computem+temsim:  E. Kirkland
  • Dr. Probe Light: STEM probe and ronchigram simulation
  • Dr. Probe : J. Barthel, Ultramic. 193 (2018) p. 1-11.
  • EDM:
  • FDES:  Van den Broek et al
  • muSTEM: Hamish Brown, L. J. Allen et al
  • mulTEM: I. Lobato, et al
  • Prismatic: PRISM and multislice, DaCosta et al , Micron 151 (2021)  p. 103141
  • QSTEM:   Quantitative TEM/STEM Simulations
  • pyQSTEM :  python interface for QSTEM (Jacob Madsen)
  • Ronchigram:  Suk Hyun Sung & Noah Schnitzer
  • STEM_CELL: HPC parallel computing, V. Grillo, E Rotunno
  • STEMsalabim: HPC parallel computing
  • STEMcl : multi-GPU code, Radek et al
  • TEM-simulator:
  • tempas: Total Resolution LLC
  • toTEM : P. J. Yuan et al, J. Micros. 287 ( 2022),  p.93-104.
Cryo EM:
  • CryoEM software: Vanderbilt
  • cryoJAX : software tools for cryoEM using JAX (M. O'Brien)
  • EMAN2: S. Ludtke, also SPARX and SPHIRE
  • EMAN wiki :
  • FREEALIGN:
  • Relion:  Scheres, 3D reconstruction from 2D images
  • Simple: Elmlund and Elmlund, 3D single particle
  • Xmipp: 3D single particle
  • Software Tools for Molecular Microscopy: long list plus descriptions
  • Topaz : denoising with machine learning (Nat. Comm. 11, 5208 (2020))
Image Analysis:
  • cornell-spectrum-imager:
  • Digital Micrograph:  from Gatan
  • DIPimage : DIPimage and DIPlib for scientific image processing
  • GENFIRE and GENFIRE-2 : 3D Image Reconstruction, from John Miao
  • HyperSpy :  multidimensional data analysis
  • ImageMagick : many format conversions, and some processing
  • IMOD: tomographic reconstruction
  • openNCEM : python EM data analysis from NCEM (Peter Ercius and others)
  • pycotem : python package to analysis crystal orientations in TEM (J. Micros.  282,2021, p.84)
  • py4DSTEM : process and analysis 4D STEM data (B. Savitzky et al MM 27(2021) p.712-743)
  • ptypy : ptychonaut, python programs by P.Thibault, B. Enders, et al
  • tomviz:  tomography
  • WhatEELS : analzes EELS images (Blanco-Portals et al, Ultramic. vol. 232, 2022, p.113403)
  • adorym : x-ray ptychography
  • fold_slice : Yi Jang's Matlab code for X-ray/electron ptychography
  • PtychoShelves_EM :  multislice electron ptychography (Z. Chen Y. Jiang, D. Muller and M. Odstrcil)
  • ptypy : Ptychography for Python
Visualization:
  • Chimera: Extensible Molecular Modeling System
  • Jmol: java 3D molecule viewer (similar to RasMol)
  • PyMol: Molecular Visualization
  • RasMol and OpenRasmol: Molecular Graphics Visualization Tool (also RasMol Classic)
  • VESTA: Visualization for Electronic and Structural Analysis, 3D
  • VESTA-3:
Materials Structure Information:
  • Crystallography Open Database :  non-biological
  • CSD : Cambridge Structural Database
  • The Electron Microscopy Data Bank : (biological) cryoEM maps and tomograms
  • Inorganic Structure Database :
  • manipulatt : rotate primitive cell
  • Materials Project : database 
  • Materiae :
  • Protein Data Base : biological
  • EELS Atlas : edge energies
  • WEB-EELS : Muller group spectra
  • NIST Ref. 64 : electron elastic scattering cross section database
Trial Data Sets:
  • Empiar :  the Electron Microscopy Public Image Archive; raw images for 3D cryo-EM
  • Warwick EM datasets : for training machine learning, Mach. Learn.: Sci. Technol. 1 (2020) 045003
Misc.:
  • AtomAI :  Deep Learning for Microscopy (Ziatdinov, Gfosh, Wong, Kalinin, arXiv:2105.07485)
  • AtomSegNet : (Houlin Xin)
  • CrystalMaker: generate crystal structures
  • ConvertXYZ : convert xyz file format between computem and vesta versions
  • eLabFTW : electronic lab notebook
  • emClarity : cryoET
  • Flexible Atomic Code :
  • LiberTEM : distributed processing of large data sets
  • Nion Swift :
  • pyxem : 4D STEM, extension of  hyperspy
  • SerialEM:  tilt series acquisition
  • STEMTool : python to analyse EM data; Mukherjee and Unocic, MM 26S2(2020)p.2960
  • TEM ImageNet : machine learning (Houlin Xin)
  • useTEM : LeBeau Group
  • hdf5 : large data file software library
  • EMD :  hdf5 based file format for Electron Microscopy Data (used by py4DSTEM)
  • NeXus : hdf5 based format for X-ray type data
  • Noise2Noise : denoising with machine learning
